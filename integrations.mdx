## Webhook Setup

Set up webhooks to receive real-time events from Successpro, such as project updates or invoice payments.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Dashboard > Integrations > Webhooks. Click "New Webhook".
  </Step>
  <Step title="Configure Endpoint" icon="link">
    Enter your endpoint URL, e.g., `https://your-webhook-url.com/successpro`.
    
    Select events like `project.updated` or `invoice.paid`.
  </Step>
  <Step title="Verify and Test" icon="check-circle">
    Click "Test" to send a sample payload. Check your logs for the event.
  </Step>
  <Step title="Activate" icon="play">
    Toggle the webhook to active. Monitor deliveries in the dashboard.
  </Step>
</Steps>

## API Endpoints Overview

Successpro's REST API uses HTTPS requests to `https://api.example.com/v1`. Authenticate with `Authorization: Bearer {YOUR_API_KEY}`.

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token for authentication.
</ParamField>

<ParamField query="project_id" param-type="string" required="false">
  Filter by specific project ID.
</ParamField>

Example: Fetch projects.

<CodeGroup tabs="JavaScript,cURL,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/projects', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  });
  const projects = await response.json();
  console.log(projects);
  ```
  ```bash
  curl -H "Authorization: Bearer YOUR_API_KEY" \
       https://api.example.com/v1/projects
  ```
  ```python
  import requests
  headers = {'Authorization': 'Bearer YOUR_API_KEY'}
  response = requests.get('https://api.example.com/v1/projects', headers=headers)
  projects = response.json()
  print(projects)
  ```
</CodeGroup>

<Response tabs="200">

## Data Sync Configurations

Configure data synchronization for bidirectional updates between Successpro and external tools.

<Tabs>
  <Tab title="CRM Sync (Salesforce)" icon="database">
    <Steps>
      <Step title="Connect Account">
        In Integrations > CRM, select Salesforce and authorize OAuth.
      </Step>
      <Step title="Map Fields">
        Map `project.name` to `Opportunity.Name` and `financials.revenue` to `Amount`.
      </Step>
      <Step title="Schedule Sync">
        Set to hourly. Test with a sample project.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Accounting Sync (QuickBooks)" icon="file-text">
    Use Zapier or direct API.
    
    ```javascript
    // Sample QuickBooks sync trigger
    await fetch('https://api.example.com/v1/sync/quickbooks', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        entity: 'invoice',
        action: 'sync'
      })
    });
    ```
  </Tab>
</Tabs>

<Expandable title="Advanced Sync Options" default-open="false">
  Customize sync filters with query params like `?status=active&min_revenue>10000`.
  
  Monitor sync health in Dashboard > Integrations > Sync Logs.
</Expandable>

<Callout kind="tip">
  Test all integrations in a sandbox environment first to avoid data discrepancies.
</Callout>
