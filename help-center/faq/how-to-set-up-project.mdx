## Overview

Setting up a project helps you centralize your documentation and workflow.

## Steps

1. Log in to your Successpro dashboard
2. Click **New Project** in the sidebar
3. Enter a name and optional description
4. Set visibility: Private or Team
5. Click **Create**

Your project is now ready. Add docs, tasks, or invite teammates from the project page.
