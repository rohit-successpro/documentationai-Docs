## Prerequisites

Before starting, ensure you have:

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- An email address for account verification
- Optional: Company details for project setup

<Callout kind="info">

Successpro works best with stable internet connections. Disable ad blockers if you encounter issues during signup.

</Callout>

## Sign Up for an Account

Create your Successpro account in under 2 minutes.

<Steps>
  <Step title="Visit the Signup Page" icon="external-link">
    Navigate to [https://hello.successpro.ai/signup](https://hello.successpro.ai/signup).
  </Step>
  <Step title="Enter Your Details" icon="user">
    Provide your first name, last name, email, and password. Accept the terms of service.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification email from Successpro and click the link to activate your account.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the login page at [https://app.successpro.ai/login](https://app.successpro.ai/login) and sign in with your credentials.
  </Step>
</Steps>

## Access and Tour the Dashboard

Once logged in, you land on the Successpro dashboard. Key sections include:

- **Projects**: Manage your technology services projects
- **Financials**: Track profitability and budgets
- **People**: Handle team assignments and visibility

<Callout kind="tip">

Book a demo via the dashboard for personalized guidance: [Book a Demo](https://www.successpro.ai/demo).

</Callout>

## Create Your First Project

Set up a sample project to test core features.

<Steps>
  <Step title="Navigate to Projects" icon="folder">
    Click "New Project" from the dashboard sidebar.
  </Step>
  <Step title="Fill Project Details" icon="edit-3">
    Enter name: "Sample Tech Services Project", description: "Test profitability tracking", and budget: `$50,000`.
  </Step>
  <Step title="Assign Team" icon="users">
    Add yourself as project manager and save.
  </Step>
</Steps>

For API-driven project creation:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.successpro.ai/v1/projects', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'API Test Project',
      budget: 50000
    })
  });
  const project = await response.json();
  console.log(project);
  ```
  ```python
  import requests

  response = requests.post(
    'https://api.successpro.ai/v1/projects',
    headers={
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    json={
      'name': 'API Test Project',
      'budget': 50000
    }
  )
  project = response.json()
  print(project)
  ```
</CodeGroup>

<ParamField path="projects" param-type="POST" required="true">
  Endpoint to create projects.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token obtained from dashboard settings.
</ParamField>
